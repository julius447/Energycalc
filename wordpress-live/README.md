# WordPress (live) — Energikalkylatorn

The **deployed** WordPress FluentSnippets version of this tool, live on
[ampy.se/energikalkylator/](https://ampy.se/energikalkylator/). Generated by converting the
standalone `vB/` build per `docs/06-FLUENTSNIPPETS-implementation.md` (namespaced `.ampy-ek`,
rem→px, `@media`→`@container`, self-hosted fonts) and wiring the calculator backend pattern
(shortcode `[ampy_ek]`, inline data, honeypot REST lead route `/wp-json/ampy-ek/v1/lead/{id}`,
post-metabox editability for the calc data + webhook + lead log).

| File | FluentSnippets type | Run at |
|------|--------------------|--------|
| `60-ampy-energikalkylator.php` | CSS | wp_head (load_as_file) |
| `61-ampy-energikalkylator.php` | PHP | everywhere |
| `62-ampy-energikalkylator.php` | JS  | wp_footer (load_as_file) |

Fonts: the site's self-hosted Outfit variable font. Trust images: `/wp-content/uploads/ampy-ek/`.
